# 2.4 Search Prefix

Write a Ruby program to search which items start with specified string of a given array.

### Input 1
```
["abcde", "abdf", "adeab", "abdgse", "bdefa", "bacdef"]
Starts with: ab
```
### Output 1
```
["abcde", "abdf", "abdgse"]
```

### Input 2
```
["abcde", "abdf", "adeab", "abdgse", "bdefa", "bacdef"]
Starts with: b
```
### Output 2
```
["bdefa", "bacdef"]
```
